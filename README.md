# Knowledge

Distilled learnings from conversations. No history — only extracted knowledge.

## Structure

```
/topics/<topic>.md   — knowledge by domain
/snippets/           — reusable code patterns
```

## Format (per entry)

```
## <Title>

<Fact or insight in 1–3 sentences.>

<!-- optional: code block, command, or example -->
```

No dates, no attribution, no context — just the knowledge itself.
